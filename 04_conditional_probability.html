Statistical independence

\[P(A \cap B) = P(A)P(B)\]

\[P(A \cup B) = P(A) + P(B) - P(A \cap B)\] \[\Rightarrow P(A \cup B) = P(A) + P(B) - P(A)P(B)\]

Independence of multiple events

The events \(E_1\), \(E_2\),…,\(E_n\) are independent if and only if \[P(E_1 \cap E_2 \cap ... \cap E_n)=P(E_1) \times P(E_2) \times ... \times P(E_n)\]

Conditional probability

\[P(A \cap B)\]

\[P(B|A)=\dfrac{P(A \cap B)}{P(A)}\]

Multiplication rule

\[P(A \cap B) = P(B|A)P(A)=P(A|B)P(B)\]

Rules for conditional probabilities

Addition and multiplication rules work the same way

\[P(E_1 \cup E_2 | A) = P(E_1|A) + P(E_2|A) - P(E_1 \cap E_2 | A)\]

\[P(E_1 \cap E_2 | A) = P(E_1|E_2|A)P(E_2|A)\]
